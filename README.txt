Clean Admin UI
---------------
Files included:
- admin/ (main admin zone)
    - dashboard.php
    - admin_login.php (login form)
    - logout.php
    - includes/
        - header.php
        - sidebar.php
        - footer.php
    - assets/
        - css/style.css
        - js/script.js
    - code/
        - admin_login.php (simple placeholder for authentication)
HOW TO USE:
- Place the 'admin' folder in your webroot.
- Adjust database and authentication code in code/admin_login.php as needed.
- Open /admin/admin_login.php to view the login page and /admin/dashboard.php for the dashboard.
