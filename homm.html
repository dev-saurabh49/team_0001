How it works

Admin Flow

  1. Admin panel में नया member create करें → system auto Team ID generate करेगा।
  2. Member को ID + अस्थायी password भेजें (SMS/WhatsApp/Email)।
  3. Announcements पोस्ट करें और tasks assign करें।
  4. Activity/attendance रिपोर्ट देखें, points & badges manage करें।

Member Flow

  1. Admin से मिली Team ID और अस्थायी password से first login करें।
  2. Password reset करें और profile update करें।
  3. Dashboard में announcements, tasks, files और points देखें।
  4. काम पूरा होने पर task complete करें और active रहें।

FAQ

क्या public registration है?

नहीं। IDs सिर्फ admin assign करता है — discipline और security के लिए।

Login routes क्या होंगे?

/login (members) और /admin (admins)। Backend में roles से guard करें।

Data privacy कैसे रखते हैं?

Role-based access, audit logs, और read-only announcements — कोई spam नहीं।

Future modules कैसे add होंगे?

Modular architecture — attendance, polls, feedback जैसे sections later plug-in.
